HTML and CSS Tables Cheat Sheet

HTML Table Tags
Tag Name Tag Description
Table <table> Defines a table
Table Row <tr> Defines a row in a table
Table Header <th> Defines a header cell in a table
Table Data <td> Defines a cell in a table
Table Caption <caption> Defines a table caption
Table Head <thead> Groups the header content in a table
Table Body <tbody> Groups the body content in a table
Table Footer <tfoot> Groups the footer content in a table
CSS Properties for Tables
Property Name Example Description
border border: 1px solid black; Sets the border for table cells
border-collapse border-collapse: collapse; Specifies whether table borders should collapse into a single border or be separated
padding padding: 8px; Sets the padding inside table cells
text-align text-align: left; Sets the horizontal alignment of text in table cells
background-color background-color: #f2f2f2; Sets the background color of table cells
caption-side caption-side: top; Specifies the position of the table caption
width width: 100%; Sets the width of the table
height height: auto; Sets the height of the table
